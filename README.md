# college-gui-chatbot
Beginner-friendly Python GUI chatbot for college helpdesk built using Tkinter. Handles student queries like admission, fees, exams, and library timing through an interactive chat interface with buttons.

# 🎓 College GUI Chatbot

A beginner-friendly Python GUI chatbot for college helpdesk made using Tkinter.

This chatbot helps students with common queries like:

- Admission
- Fees
- Exam Dates
- Library Timing
- College Timings
